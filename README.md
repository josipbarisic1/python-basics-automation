Python Basics & Automation

Collection of Python automation scripts built while learning practical automation workflows.

Structure:

basics/
Learning exercises covering Python fundamentals:
- Day 1 - number loop script
- Day 2 - user list processing
- Day 3 - file processing
- Day 4 - CSV filtering
- Day 5 - API request and JSON export
- Day 6 - CSV enrichment with API data
- Day 7 - refactoring CSV automation script and adding basic error handling

automation/
Production-ready automation tools:
- CSV Cleaner - data cleaning and deduplication
- CSV Merger - combine multiple CSV files
- File Organizer - automatic file sorting by type
- Web Scraper - extract and process web data
- Folder Monitor - automated file processing pipeline

Each automation project includes its own README and test files.
